Inktober Icons Challenge

Graphics / Figma / Daily Challenge

Combining design, time management, and vector skill practice, I participated in Inktober by creating daily icons in Figma.

Inktober is a global challenge where artists create a daily ink drawing throughout October, fostering creativity and inking skills.I sketched icon ideas daily, then compiled and laid out a poster to showcase my Inktober challenge outcome.
